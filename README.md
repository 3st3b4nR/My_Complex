# My_Complex
Software de gestion para conjuntos residenciales

En este primer entregable hemos creado una autenticación de usuarios para el login con valores simulados debido a que aún no contamos con conexión a la base de datos. Además, creamos un controlador de ver apartamentos para que los usuarios correspondientes (Residente y propietario) puedan verlos y en caso tal de no pertenecer al rol, va a generar un error  403(Forbidden).

--Paquetes necesarios para la solución:
Microsoft.IdentityModel.Tokens
System.IdentityModel.Tokens.Jwt
Swashbuckle.AspNetCore
